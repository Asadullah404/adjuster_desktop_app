SMMC - LabReports With Watermark v1.0.0
==================================================
FEATURES:
---------
✓ Adjustable top margin (1-10 lines)
✓ Watermark opacity control (0.1-1.0)
✓ Customizable watermark positioning
✓ Support for multiple image formats (PNG, JPG, JPEG, GIF, BMP, TIFF)
✓ Process & Open (temporary preview)
✓ Process & Save (permanent output)
✓ Progress tracking with status updates
✓ Automatic cleanup of temporary files

SYSTEM REQUIREMENTS:
-------------------
• Windows 10 or later (32-bit or 64-bit)
• 100MB available disk space
• Administrator privileges for installation
• PDF viewer (Adobe Reader, Edge, Chrome, etc.)

INSTALLATION:
------------
1. Run the installer as Administrator
2. Follow the setup wizard instructions
3. Choose installation directory (default recommended)
4. Select optional shortcuts and context menu integration
5. Click Install and wait for completion

USAGE:
------
1. Launch SMMC from Start Menu or Desktop
2. Click "Browse" to select your input PDF file
3. Optionally select a logo image for watermarking
4. Adjust settings:
   - Number of lines for top margin
   - Watermark opacity
   - Starting line for watermark placement
5. Choose action:
   - "Process & Open PDF": Creates temporary file for preview
   - "Process & Save PDF": Saves processed PDF permanently


TROUBLESHOOTING:
---------------
• If the app doesn't start, try running as Administrator
• For large PDFs, processing may take several minutes
• Ensure input PDF files are not password-protected
• Close PDF viewers before processing the same file again
• Default logo.png is automatically detected if placed in app folder
